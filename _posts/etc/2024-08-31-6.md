# _variables.scss 파일 수정
<br/>
/_sass/minimal-mistakes/_variables.scss 파일로 이동한다.
<br/>
x-large 값을 수정해 준다.
<br/>
```
/*
   Breakpoints
   ========================================================================== */

$small: 600px !default;
$medium: 768px !default;
$medium-wide: 900px !default;
$large: 1024px !default;
$x-large: 1500px !default;  //원래값 : 1280px
$max-width: $x-large !default;
```

# 릿코드 문제풀이

** 작업중

```
String s = "LVIII";
int sum = 0;
char[] romanArr = {'I','V','X','L','C','D','M'};
Map<Character, Integer> romanMap = new HashMap<Character, Integer>(){
    {
        put('I', 1);
        put('V', 5);
        put('X', 10);
        put('L', 50);
        put('C', 100);
        put('D', 500);
        put('M', 1000);
    }
};
for (int i = 0; i < s.length(); i++) {
    if(i != s.length()-1) {
        char thisChar = s.charAt(i);
        char nextChar = s.charAt(i+1);
        int thisIdx = Arrays.asList(romanArr).indexOf(thisChar);
        int nextIdx = Arrays.asList(romanArr).indexOf(nextChar);
        if(thisIdx < nextIdx) { //현재값이 다음값 보다 클 경우
            sum += romanMap.get(nextChar) - romanMap.get(thisChar); //다음 값에서 현재 값을 뻄
            i++; // 다음값을 이미 계산 했음으로 1 추가로 증가
        } else {
            sum += romanMap.get(thisChar);
        }
    } else {
        sum += romanMap.get(s.charAt(i));
    }
}
```

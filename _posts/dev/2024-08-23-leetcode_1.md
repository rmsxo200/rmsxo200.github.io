# 릿코드 공부

```
  public int[] twoSum(int[] nums, int target) {
        for (int i = 0; i < (int) Math.pow(10, 4); i++) {
            for (int j = 0; i < (int) Math.pow(10, 4); j++) {
                if(i != j && nums[i] + nums[j] == target) {
                    return new int[] {i, j};
                }
            }
        }

        return  null;
  }
```
